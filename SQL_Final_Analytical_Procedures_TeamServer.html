# create clean text function

def clean_text(t):
    
    text = ''.join(x for x in t if x not in string.punctuation)
    
    tokens = re.split('\W+', text)
    
    text = [wn.lemmatize(word.lower()) for word in tokens if word not in stops]
    
    return text

Sentiment scores¶

Using textblob package to generate sentiment scores.

First score is polarity, or degree of positive/negative sentiment.

Second score is subjectivity, or how subjective the text is.

Source: https://data-science-blog.com/blog/2018/11/04/sentiment-analysis-using-python/
